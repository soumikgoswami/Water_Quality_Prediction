The Water Quality Prediction Dashboard is a data visualization project developed using Power BI to analyze and interpret various water quality parameters such as pH, hardness, solids, chloramines, sulfate, conductivity, organic carbon, trihalomethanes, and turbidity. The goal of the project is to determine whether water is potable or non-potable by analyzing these parameters and presenting the insights through interactive dashboards. The system uses Power Query for data cleaning, DAX for creating measures, and Power BI visualizations such as KPI cards, pie charts, bar charts, and scatter plots to display the results. This dashboard helps users easily explore water quality trends, understand parameter relationships, and make better decisions regarding water safety.
